COMO HACER UN MANUAL DE USUARIO

1. se debe tener en cuenta el objetivo del manual

2. se debe tener conocimientos basicos de informatica, sistemas operativos, tener conocimientos de politicas de seguridad

3. se debe tener las especificaciones técnicas del hadware y software, tambien se debe aclarar si hay limitaciones en cuanto al sistema operativo

4. se debe explicar detalladamente la forma de ingresar al sistema de infomacion, con cada uno de los usuarios que se tenga completados

5. se debe explicar el mapa de navegacio, la descripción del miso y la descripción paso a paso de cada uno de los formularios que se manejan en el aplicativo

6. se debe presentar de manera detallada, el mapa de navegación con sus respectivos botones y funciones

7. se debe describir el paso a paso los procesos, con sus respectivas pantallas, se recomienda incorporar algunos ejemplos
